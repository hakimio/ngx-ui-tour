ngx-ui-tour is a UI tour library built for Angular. It's inspired by angular-ui-tour.

TourNgBootstrapModule is an implementation of the tour UI that uses NgBootstrap popovers to display tour steps.

Installation
  1. npm install ngx-ui-tour-ng-bootstrap bootstrap @popperjs/core @ng-bootstrap/ng-bootstrap
  2. Import TourNgBootstrapModule into your app module If you are using v11 or older, you have to import TourNgBootstrapModule.forRoot()
  3. Make sure RouterModule is imported in your app module
  4. Add bootstrap SCSS to your styles.scss:
  5. Make sure NG Bootstrap is installed correctly
