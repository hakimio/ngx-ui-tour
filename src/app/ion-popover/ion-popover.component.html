ngx-ui-tour is a UI tour library built for Angular. It's inspired by angular-ui-tour.

TourIonPopoverModule is an implementation of the tour UI that uses Ionic Popover to display tour steps.

Installation
  1. npm install ngx-ui-tour-ionic @ionic/angular
  2. Import TourIonPopoverModule into your app module
  3. Make sure RouterModule is imported in your app module
  4. Make sure Ionic is installed correctly
