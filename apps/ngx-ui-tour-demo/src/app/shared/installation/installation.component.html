Installation
This guide is for v{{version}}. Older version of the documentation is available here.
  1. npm install {{ packageName() }}
  2. Add provideUiTour() to the list of providers in your app.config.ts. Add provideUiTour() to the list of providers in your app.module.ts.
  3. Make sure Angular Router is enabled using provideRouter(appRoutes) or RouterModule.forRoot(appRoutes) .
