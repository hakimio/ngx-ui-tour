Usage
  1. Import {{ componentName() }} in your standalone component
  2. Add {{ tourStepTag }} to your root app component's template (once in the whole app)
  3. Define anchor points for the tour steps by adding the tourAnchor directive throughout your app.
    You have to import {{ componentName() }} in all your standalone components where you use tourAnchor directive.
  4. Define your tour steps using tourService.initialize(steps)
  5. Start the tour with tourService.start()
  6. Check out the demo source code for an example.
