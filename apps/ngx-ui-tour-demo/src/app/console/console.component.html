ngx-ui-tour is a UI tour library built for Angular. It's inspired by angular-ui-tour.

TourConsole is a minimal implementation of the tour UI that uses only the browser's JavaScript console to display tour steps.

Use left/right arrow keys to navigate through the tour and escape key to end the tour.

 Installation
  1. npm install ngx-ui-tour-console
  2. Import TourConsoleModule into your app module
    If you are using v9 or older, you have to import TourConsoleModule.forRoot()
  3. Make sure RouterModule is imported in your app module
