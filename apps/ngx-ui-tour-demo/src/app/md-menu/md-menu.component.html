ngx-ui-tour is a UI tour library built for Angular. It's inspired by angular-ui-tour.

TourMatMenuModule is an implementation of the tour UI that uses Angular Material MatMenu to display tour steps.

Installation
  1. npm install ngx-ui-tour-md-menu @angular/material @angular/cdk
  2. Import TourMatMenuModule into your app module If you are using v9 or older, you have to import TourMatMenuModule.forRoot()
  3. Make sure RouterModule is imported in your app module
  4. Make sure Angular Material is installed correctly
