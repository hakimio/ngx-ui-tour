ngx-ui-tour is a UI tour library built for Angular. It's inspired by angular-ui-tour.

TourTuiHintModule is an implementation of the tour UI that uses Taiga UI Hint to display tour steps.

Installation
  1. npm install ngx-ui-tour-tui-hint @taiga-ui/core @taiga-ui/cdk @taiga-ui/icons
  2. Import TourTuiHintModule into your app module If you are using v3 or older, you have to import TourTuiHintModule.forRoot()
  3. Make sure RouterModule is imported in your app module
  4. Make sure Taiga UI is installed correctly
